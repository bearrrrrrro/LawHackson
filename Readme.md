# 法律黑客松

## 事前準備


### `.env`

如果要用到外面的 API 等需要金鑰的東西，請把金鑰放在 `.env` 中。

### 安裝套件

把 `requirements.txt` 裡面的東西裝一裝，可以執行 `pip install -r requirements.txt` 他就會幫你裝完

## Usage

1. 把要跑的判決 `json` 塞進 `input` 資料夾
2. 執行 `python main.py`
3. 等一下，我還不知道要等多久，應該很快拉
4. 輸出的檔案在 `result.csv`
